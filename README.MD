
## [ElectroShop] 프로젝트란?
본 프로젝트는 스프링 3.2 버전을 연습하기위해 만든 쇼핑몰 프로토타입 입니다. 
데이터베이스단은 마이바티스와 오라클11g_ex을 이용하였습니다.
기본적인 스프링 mvc패턴을 이용하였으며 view는 tiles를 이용하였습니다.
자바스크립트단은 jQuery, angularJs, ajax를 이용하였습니다.
css는 부트스트랩을 이용하였습니다.
